PWM Output Frequencies Configuration

Step by step practical guide to configuring higher and user defined PWM output frequency on Arduino. Understand how microcontrollers generate PWM output signals on fundamental level Configure 8bit PWM timer for higher preset PWM output frequencies. Configure 16bit PWM timer for user defined higher PWM output frequencies. Demonstrate the concept using PWM controlled DC motor example.

Step-by-step video guide

In this video guide, you will learn how to configure higher or custom PWM output frequencies on Arduino . Arduino Nano is used to perform PWM voltage control at higher PWM output frequencies. Video covers the fundamentals of PWM signal generation by microcontrollers and manipulating PWM registers to get desired output frequencies. Please watch the whole video for better understanding.

Wiring Diagram

Below is the wiring diagram for the components used in this project. Example1 uses PWM pin5 with 8but PWM timer. Example2 uses PWM pin9 with 16bit PWM timer. Pin7 is used to set motor direction. Make sure there's a common ground between the motor driver, arduino and external power supply.
